# NestJs-Template

Service catered to allocate resources against potential opportunities

## Installation

```bash
$ npm install
```

## Configration

Create a .env file in root directory
refer to .env.example file
Add database password

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
